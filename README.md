## Run The App Locally

1. Clone the repo
   ```sh
   git clone https://github.com/mosiej803/crypto-calc-app.git
   ```
2. Install Live Server
   ```sh
   npm i live-server
   ```
3. Run local server
   ```sh
   live-server
   ```
